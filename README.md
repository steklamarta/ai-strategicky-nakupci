# AI Strategický Nákupčí – Demo projekt

Tento repozitář obsahuje demonstrační implementaci multi-agentního AI systému
pro strategické vyhodnocování nabídek ve výběrovém řízení.

Projekt slouží ke studijním a prezentačním účelům.

## Agenti
- **Strategický nákupčí** – analyzuje výzvu a nabídky pomocí dílčích rolí
- **Školitel** – vysvětluje výstupy strategického nákupčího srozumitelně pro začátečníky

## Obsah repozitáře
- `/prompts` – systémové prompty jednotlivých agentů
- `/data` – ukázková výzva a nabídky dodavatelů
- `/api` – jednoduché (fake) API simulující komunikaci s LLM
